# TD1 ENSISA – Intelligence Artificielle : Markov Decision Process

Ce dépôt contient la résolution du problème n°2 du TD1 d'IA réalisé à l’ENSISA, portant sur les processus de décision markoviens (MDP).

## Contenu
- `TD1_Pb2_IA_ValueIteration.py` → Implémentation de la Value Iteration en Python
- `TD1_Pb2_IA_PolicyIteration.py` → Implémentation de la Policy Iteration en Python

Les deux scripts résolvent un environnement de type GridWorld 4×4 :
- Récompense : -1 par pas  
- États terminaux : (0,0) et (3,3)  
- Facteur d’actualisation (γ) : 1, 0.9 et 0.8
